# Pizza-Sales-Dashboard
Pizza Sales Performance Analytics Dashboard
An interactive Power BI dashboard that analyzes pizza sales performance, helping identify revenue trends, busiest days and times, top‑selling categories, and order patterns to support data‑driven decisions for a pizza store.

Tech Stack
Power BI Desktop – 
    Data modeling, DAX measures, and interactive visualizations
SQL / Excel / CSV –
    Source data storage and basic preprocessing (depending on what you actually used)
DAX (Data Analysis Expressions) –
    Calculated measures for KPIs like total revenue, average order value, and pizzas per order


Features / Highlights
Executive KPI overview
Cards for Total Revenue, Average Order Value, Total Pizzas Sold, Total Orders, and Average Pizzas per Order.
Time‑based sales analysis
Daily and monthly order trends to understand seasonality and peak demand periods.
Busiest days and times
Insights showing which days of the week and which months have the highest order volumes, highlighting Friday/Saturday evenings and key months.
Category and size performance
Donut and bar charts showing percentage of sales and total pizzas sold by pizza category (Classic, Supreme, Veggie, Chicken, etc.) and by pizza size (Small/Medium/Large/X‑Large).
Best/Worst sellers page (second report page)
Dedicated view to compare top‑performing and under‑performing pizzas based on revenue and quantity sold, helping optimize the menu.
